## Flake Budget and Rerun Policy

- PHPUnit flake budget: `0` (no retries).
- Playwright flake budget: `1` automated retry in CI only.
  - Configured in `playwright.config.ts` with `retries: process.env.CI ? 1 : 0`.
- Policy:
  - A run is accepted only if final CI status is green after permitted retry budget.
  - Repeated flaky behavior should be promoted to a bug and tracked before release sign-off.
