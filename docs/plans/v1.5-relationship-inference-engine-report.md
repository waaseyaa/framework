# v1.5 Relationship Inference Engine Report (#166)

## Goal
Infer deterministic, review-safe relationship candidates from ingested text and map them into the existing relationship contract.

## Delivered
- Added `RelationshipInferenceEngine`:
  - deterministic pair traversal over sorted node keys,
  - lexical overlap inference (`>=2` shared normalized tokens),
  - explicit-relationship pair dedupe,
  - review-safe inferred edges (`status=0`, `inference_review_state=needs_review`).
- Added opt-in ingestion wiring:
  - new `ingest:run --infer-relationships` flag,
  - inferred relationships merged before validation gate pass,
  - `meta.inference_enabled` and `meta.inferred_relationship_count`.
- Added inference diagnostics:
  - `inference.relationship_inferred` entries with confidence/source/review metadata.
- Added contract spec:
  - `docs/specs/relationship-inference-contract.md`.
