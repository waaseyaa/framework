## Scope delivered
- Implemented public SSR cache-control profile suitable for CDN integration in `HttpKernel`:
  - `max-age`
  - `s-maxage`
  - `stale-while-revalidate`
  - `stale-if-error`
- Added SSR cache profile config resolution for deterministic tuning without contract drift:
  - `ssr.cache_max_age`
  - `ssr.cache_shared_max_age`
  - `ssr.cache_stale_while_revalidate`
  - `ssr.cache_stale_if_error`
- Added deterministic surrogate headers on cacheable public SSR entity responses:
  - `Surrogate-Key` (entity/workflow/view/lang/graph scopes)
  - `X-Waaseyaa-Render-Variant`
  - `X-Waaseyaa-Render-Workflow`
- Ensured surrogate headers are emitted consistently for both cache-hit and cache-miss SSR paths.
- Updated infrastructure spec to document the v1.4 public SSR/CDN contract and invalidation semantics.
