- [ ] **Step 3: Write minimal FrameworkExtension with url() function**

```php
<?php

declare(strict_types=1);

namespace Waaseyaa\SSR\Twig;

use Symfony\Component\Routing\Generator\UrlGeneratorInterface;
use Twig\Extension\AbstractExtension;
use Twig\TwigFunction;
use Waaseyaa\Routing\WaaseyaaRouter;

final class FrameworkExtension extends AbstractExtension
{
    public function __construct(
        private readonly WaaseyaaRouter $router,
        private readonly string $currentPath = '/',
    ) {}

    /** @return TwigFunction[] */
    public function getFunctions(): array
    {
        return [
            new TwigFunction('url', $this->url(...)),
            new TwigFunction('path', $this->path(...)),
            new TwigFunction('entity_url', $this->entityUrl(...)),
            new TwigFunction('is_front', $this->isFront(...)),
            new TwigFunction('current_path', $this->currentPath(...)),
        ];
    }

    /**
     * Generate an absolute URL for a named route.
     *
     * @param array<string, mixed> $parameters
     */
    public function url(string $name, array $parameters = []): string
    {
        return $this->router->generate($name, $parameters, UrlGeneratorInterface::ABSOLUTE_URL);
    }

    /**
     * Generate a relative path for a named route.
     *
     * @param array<string, mixed> $parameters
     */
    public function path(string $name, array $parameters = []): string
    {
        return $this->router->generate($name, $parameters);
    }

    /**
     * Generate the canonical URL for an entity.
     */
    public function entityUrl(string $entityType, string|int $id): string
    {
        return $this->router->generate($entityType . '_view', ['id' => $id]);
    }

    public function isFront(): bool
    {
        return $this->currentPath === '/';
    }

    public function currentPath(): string
    {
        return $this->currentPath;
    }
}
```

```php
<?php

declare(strict_types=1);

namespace Waaseyaa\Testing\Traits;

use Twig\Environment;
use Twig\Loader\ArrayLoader;

trait InteractsWithRenderer
{
    /**
     * Render a Twig template string with the given context.
     *
     * @param array<string, mixed> $context
     */
    protected function renderTemplate(string $template, array $context = []): string
    {
        $twig = new Environment(new ArrayLoader([
            '__test__' => $template,
        ]));

        return $twig->render('__test__', $context);
    }

    /**
     * Assert that rendered HTML contains the given string.
     */
    protected function assertRenderedContains(string $expected, string $html, string $message = ''): void
    {
        $this->assertStringContainsString(
            $expected,
            $html,
            $message ?: "Expected rendered output to contain '{$expected}'.",
        );
    }

    /**
     * Assert that rendered HTML does NOT contain the given string.
     */
    protected function assertRenderedNotContains(string $unexpected, string $html, string $message = ''): void
    {
        $this->assertStringNotContainsString(
            $unexpected,
            $html,
            $message ?: "Expected rendered output to NOT contain '{$unexpected}'.",
        );
    }

    /**
     * Assert that rendered HTML contains the given element tag.
     */
    protected function assertRenderedElementExists(string $tag, string $html, string $message = ''): void
    {
        $this->assertMatchesRegularExpression(
            '/<' . preg_quote($tag, '/') . '[\s>]/i',
            $html,
            $message ?: "Expected rendered output to contain <{$tag}> element.",
        );
    }
}
```

## Codebase Conventions (for workers)

- `final class` by default for concrete implementations
- PHPUnit 10.5 attributes: `#[Test]`, `#[CoversClass(...)]`, `#[CoversNothing]` for integration/template tests
- Namespace: `Waaseyaa\PackageName\` (e.g., `Waaseyaa\SSR\Twig\`)
- Named constructor parameters
- Twig extension pattern: extend `AbstractExtension`, return `TwigFunction[]` from `getFunctions()`
- Existing extension example: `packages/search/src/Twig/SearchTwigExtension.php`
- PR title format: `feat(#N): description`
- No `psr/log` — use `error_log()` for best-effort logging
